# 📊 Sales Performance Dashboard (Excel)

## 📌 Overview
This project presents an interactive sales dashboard built using Microsoft Excel. It analyzes sales performance, profit trends, and business insights across different cities, categories, and time periods.

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Dashboard Design  

---

## 📂 Project Features
- Data cleaning and transformation  
- Calculated columns (Profit Margin, Year, Month)  
- KPI metrics (Total Sales, Total Profit, Total Orders, Average Sales)  
- Pivot table analysis (City-wise, Category-wise, Monthly trends)  
- Interactive dashboard with charts and visuals  

---

## 📈 Key Insights
- Hyderabad has the highest sales among all cities  
- December shows peak sales performance  
- Clothing category generates the highest profit  
- Sales trend fluctuates across months  
- Mumbai and Hyderabad are top-performing cities
